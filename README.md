## Описание программы


**Random Password Generator** — это GUI-приложение для генерации случайных паролей с возможностью настройки их сложности. Приложение позволяет выбирать длину пароля, типы используемых символов (цифры, буквы, спецсимволы), сохранять историю сгенерированных паролей в JSON-файл и загружать её обратно.


### Возможности:
- ✅ Генерация случайных паролей заданной длины (4–64 символа)

- ✅ Выбор типов символов: цифры, буквы, спецсимволы

- ✅ Ползунок для удобной настройки длины пароля

- ✅ Сохранение истории паролей в JSON-файл

- ✅ Загрузка истории из JSON-файла

- ✅ Копирование пароля в буфер обмена

- ✅ Очистка всей истории

- ✅ Валидация ввода (минимальная/максимальная длина, выбор хотя бы одного типа символов)

## Примеры использования


### Генерация пароля:

- Настройте длину пароля с помощью ползунка

- Отметьте нужные типы символов (цифры, буквы, спецсимволы)

- Нажмите кнопку "Сгенерировать пароль"

- Пароль появится в поле ввода и добавится в историю

- Сохранение и загрузка:
- Автоматически: История сохраняется в файл password_history.json после каждой генерации

- Нажмите "Сохранить историю" — данные сохранятся в тот же файл вручную

- При запуске приложения история автоматически загружается из файла

- Нажмите "Очистить историю" — все записи удаляются

- Копирование пароля:
- Нажмите "Копировать в буфер" — сгенерированный пароль скопируется в буфер обмена

- Валидация данных
- Приложение проверяет:

- Длина пароля не менее 4 и не более 64 символов

- Должен быть выбран хотя бы один тип символов (цифры, буквы или спецсимволы)
